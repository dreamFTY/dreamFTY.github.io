无需软件互传、拒绝缓慢同步｜打破Windows与Apple的文件同步局限

发表于 2023-12-23 更新于 2023-12-28 分类于 效率与工具 ， 同步与协同 阅读次数：
无需使用聊天软件的文件互传，远离缓慢的同步软件，利用局域网在Windows与Apple产品间实现快捷迅速的文件互传

前言

但我们需要在 Windows 和 Apple 设备间传输文件时，许多人会选择利用微信、QQ等聊天软件来发送文件，还有些人会利用 Windows内嵌的 iCloud进行同步。然而，前者的步骤虽然简单，却过于繁杂，需要打开软件，找到聊天窗口，找到文件，长按……；后者过于“抽风”，同步的速度时快时慢，有时候甚至还会直接“罢工” 。而作为一个双平台用户，我既不想下载多余的软件，更不想被同步的速度所困扰，那究竟怎样才能实现「无缝衔接」？

利用 Windows 下的“文件共享”

  1. 在电脑任意位置新建文件夹
  2. 右键文件夹，点击属性，在菜单栏上找到“共享”
  3. 进入“高级共享”
  4. 框选“共享此文件夹”，在进入“权限”
  5. 点击“Everyone”，在“权限”中勾选“允许完全控制”或依次全部勾选


6. 点击确定，在返回的界面中点击“共享”


7. 如图，按步骤添加用户“Everyone”，并将其“权限级别”修改为“允许读写/输入”，否则在 Apple 中你将没有修改文件的权限

查看你的 ip 地址和用户名

  1. 实用快捷键win+R，输入CMD并确定


2. 在命令行中输入ipconfig，并查看自己的 IPv4 地址和用户名

在 Apple 产品中添加“本地服务器”

  1. 打开“文件”，点击左上角的三个点，选择连接服务器


2. 在“服务器”中输入smb:// + 你的ip地址


3. 输入你电脑的用户名（第二步所得）和密码（电脑开机密码，如果没有可以自行设置）

常见问题

如果在 Apple 产品中添加“本地服务器”时失败了，有以下两个解决方案：

用户名问题

  1. 按下键盘 win + R
  2. 输入 netplwiz ，点击确定
  3. 在这里你就可以看到你真正的用户名是什么

密码问题

  1. 如果修改用户名后仍然不行，先不要改回原先的用户名
  2. 在共享界面中的“密码保护”一栏点击“网络和共享中心”


3. 点击“所有网络”，关闭“密码保护的共享”


4. 在连接服务器时将“连接身份”改为客人，就可以直接连接了

有人可能会想：这样做不会让其他人都看得到我的文件内容了吗？确实是这样的，但是正如“前言”所说，这是在局域网内实现的文件同步。同样，其他人要查看你的同步文件内容，不仅要知道你的IPv4地址，还需要知道你的 Wi-Fi 密码，除非你的 Wi-Fi 密码很容易被知道或者你主动告诉他人，不然你的文件内容是不会被其他人看到的。

尾声

以上就是利用局域网实现的文件互传功能啦！如果有需要传输的文档、照片等文件，只需将其拖入共享的文件夹即可。快去试试吧！
